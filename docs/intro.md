# Introduction

`librtui` is Radxa's TUI scripting library. It is intended to provide easy-to-use
helpers to build TUI application's interface, as well as utility functions
to emulate some high level programming language features within Bash.

Currently, it is used by following projects:

* [`rsetup`](https://github.com/radxa-pkg/rsetup)
* [`rsdk`](https://github.com/RadxaOS-SDK/rsdk)
* [`radxa-oobe`](https://github.com/radxa-pkg/radxa-oobe)
